# Altrua-Kolektif
Altrua Kolektif adalah aplikasi galang dana universal berbasis altruisme—tindakan tulus menopang sesama tanpa pamrih. Mengusung transparansi radikal, platform ini menggunakan sistem Milestone dan verifikasi foto lapangan via Multer. Menghubungkan company (organisator) dan donate (donatur) secara mudah, aman, dan kredibel.
